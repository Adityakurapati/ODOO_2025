## 🚀 Project Overview

SkillSwap Arena is a revolutionary skill exchange platform that gamifies the learning experience. Users can offer their expertise in exchange for learning new skills from others, creating a vibrant community of knowledge sharing.

### ✨ Key Features

#### 🎯 **Core Functionality**
- **Profile Management**: Create detailed profiles with skills offered and wanted
- **Skill Matching**: Advanced algorithm to match users based on complementary skills
- **Swap Requests**: Send and receive skill exchange requests with custom messages
- **Rating System**: Rate and review completed skill exchanges
- **Privacy Controls**: Public/private profile settings

#### 🎮 **Gamification Elements**
- **XP System**: Earn experience points for platform activities
- **Level Progression**: Level up as you complete more skill swaps
- **Achievement Badges**: Unlock achievements for various milestones
- **Streak System**: Maintain daily activity streaks
- **Leaderboards**: Compete with other users on various metrics

#### 🛡️ **Admin Panel**
- **Content Moderation**: Review and moderate reported content
- **User Management**: Ban users who violate platform policies
- **Swap Monitoring**: Track all platform swaps and their statuses
- **Platform Messaging**: Send announcements to all users
- **Analytics & Reports**: Download detailed platform analytics

#### 🎨 **Modern UI/UX**
- **Dark Theme**: Sleek dark mode interface
- **Interactive Elements**: Smooth animations and hover effects
- **Responsive Design**: Works perfectly on all devices
- **Accessibility**: WCAG compliant design

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/Adityakurapati/ODOO_2025/
   cd ODOO_2025
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
   npm install
   \`\`\`

3. **Run the development server**
   \`\`\`bash
   npm run dev
   \`\`\`

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

\`\`\`bash
npm run build
npm start
\`\`\`

## 🎯 Problem Statement Solution

**PS #1: Skill Exchange Platform**

Our solution addresses the growing need for peer-to-peer learning by creating an engaging platform where users can:

1. **Exchange Skills Efficiently**: Match users with complementary skills
2. **Build Learning Communities**: Foster connections between learners and mentors
3. **Gamify Education**: Make learning fun and rewarding through game mechanics
4. **Ensure Quality**: Rating and review system maintains high standards
5. **Scale Globally**: Platform designed for worldwide skill exchange
